# homebrew-kosh


Homebrew tap for [kosh](https://github.com/VaarunSinha/kosh) — encrypted secret vault for developers and teams.

</div>

## Install

```sh
brew install VaarunSinha/kosh/kosh
```

That's it. Homebrew auto-taps this repo on first install.

## Manual tap

```sh
brew tap VaarunSinha/kosh
brew install kosh
```
